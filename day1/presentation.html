Welcome

Today, you become a data analyst

You don't need to be a programmer or a mathematician. If you can ask a question and read a chart, you can do data science.

By the end of Week 1, you will have collected your own data, pulled data off the web, and analyzed a real dataset — all with code you run yourself.

Magaan lang ito. Sundan nyo lang ang mga hakbang, isa-isa. Walang madaling matatalo, lahat tayo magsisimula sa zero.

The big idea

What is data science?

Turning data (raw numbers and facts) into decisions (things we actually do).

Question
→
Collect data
→
Analyze
→
Visualize
→
Decision

Every app, business, and government office runs on this loop. We'll practice the whole loop today — just with small, friendly datasets.

Why it matters

Data is everywhere

📋

Surveys

Google Forms, feedback, polls — people answering questions.

🌐

The web

News, prices, listings — public pages full of data to collect.

🏢

Records

Schools, weather, tourism — organizations tracking the real world.

The skill is the same no matter the source: collect it, clean it, understand it.

Our toolkit

What we'll use today

Google Colab — write & run code in your browser
Python — the language of data science
pandas — tables & data
matplotlib — charts
seaborn — prettier charts
BeautifulSoup — web scraping

No installs on your computer. Everything runs online in Google Colab. You just need a browser and a Google account. Tip: run each cell with Shift + Enter, top to bottom.

Activity 2

Web Scraping

Data isn't always handed to you — sometimes you collect it yourself from web pages. That's scraping.

A

Live website

Scrape today's headlines from a real news site and analyze them.

B

Local dataset

Read 3 tables from a CAR-region data file: weather, schools, tourism.

📊

Dashboard

Turn the scraped tables into a 6-chart overview.

Two honest habits: always check a site's terms before scraping, and always know if your data is real or simulated. (Ours is simulated for practice.)

Before we start

Three ground rules

  • Go in order. Run cells top to bottom with Shift + Enter.
  • Errors are normal. Read the message, fix, re-run. Tanong lang.
  • Know your data. Always ask: is this real or simulated?

You don't need to memorize the code. The goal today is to understand the flow — the rest comes with practice on Weeks 2 and 3.
